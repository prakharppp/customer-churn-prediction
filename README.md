# Customer Churn Prediction using Machine Learning

This project focuses on predicting customer churn using basic machine learning models. The goal is to identify customers who are likely to leave a service based on their usage and account information.

## Overview

Customer churn is an important problem in industries like telecom and SaaS. In this project, I worked with a customer dataset and applied classification techniques to predict whether a customer will churn or not.

## Steps Performed

- Data loading and basic exploration
- Data cleaning (handling missing values, converting data types)
- Encoding categorical variables
- Feature scaling
- Train-test split
- Model building using Logistic Regression and Decision Tree
- Model evaluation using accuracy, confusion matrix, precision and recall

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Models Used

- Logistic Regression
- Decision Tree Classifier

## Evaluation Metrics

- Accuracy
- Confusion Matrix
- Precision and Recall

## Dataset

The dataset used in this project is the Telco Customer Churn dataset, which is publicly available.

You can download it from Kaggle:
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

## Notes

This is a beginner to intermediate level project focused on understanding the machine learning workflow and applying basic classification models.
